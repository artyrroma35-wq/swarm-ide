# Топология Swarm IDE

## Два примитива
- **create** — создать нового агента
- **send** — отправить сообщение агенту или в группу

## IM система ≠ Agent Loop
- **IM система**: groups + messages, кто кому что сказал
- **Agent Loop**: llmHistory + LLM вызовы + инструменты
- Агент **НЕ ПУБЛИКУЕТ** свои мысли автоматически
- Только явный вызов send_* создаёт сообщение

## Как работает агент
1. Получает сообщение из группы
2. Добавляет в свою llmHistory
3. Вызывает LLM (может делать инструменты)
4. Результат пишет в llmHistory
5. Явно вызывает send_* для отправки

## Рекурсия
- Любой агент может create + send
- Топология строится рекурсивно
- Нет центрального контроллера
